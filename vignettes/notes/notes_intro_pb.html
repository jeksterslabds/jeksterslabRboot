Random Variable

Let \(X\) be a random variable from a normal distribution with \(\mu = 0\) and \(\sigma = 1\).

\[\begin{equation} X \sim \mathcal{N} \left( \mu = 0, \sigma^2 = 1 \right) \end{equation}\]

This data generating function is referred to as \(F\).

Realization of the Random Variable

Let \(\mathbf{x}\) be a realization of the random variable \(X\).

x <- rnorm(
  n = n,
  mean = mu,
  sd = sigma
)

Parametric Bootstrapping

\(\mathbf{x}\) which is a realization of the random variable \(X\) is used in bootstrapping. This sample data is referred to as the empirical distribution \(\hat{F}\).

In parametric bootstrapping we make a parametric assumption about the distribution of \(\hat{F}\). In the current example, we assume a normal distribution \(\hat{F}_{\mathcal{N} \left( \mu, \sigma^2 \right)}\). Data is generated \(B\) number of times (100 in the current example) from the empirical distribution \(\hat{F}_{\mathcal{N} \left( \mu, \sigma^2 \right)}\) using parameter estimates (\(\hat{\mu} = 0.0325148\) and \(\hat{\sigma}^2 = 1.0844243\)).

\(x^{*} = \left\{ x^{*1}, x^{*2}, x^{*3}, x^{*b}, \dots, x^{*B} \right\}\)

\(x^{*}\) is a set of \(B\) bootstrap samples.

x_star <- pb_univ(
  rFUN = rnorm,
  n = n,
  B = B,
  mean = muhat,
  sd = sigmahat
)

\(\hat{\theta}^{*} = \left\{ \hat{\theta}^{*} \left( 1 \right), \hat{\theta}^{*} \left( 2 \right), \hat{\theta}^{*} \left( 3 \right), \hat{\theta}^{*} \left( b \right), \dots, \hat{\theta}^{*} \left( B \right)\right\}\)

The parameter is estimated for each of the \(B\) bootstrap samples. For example, if we are interested in making inferences about the mean, we estimate the mean \(B\) times. The estimated parameters form the bootstrap sampling distribution \(\hat{\theta}^{*}\).

thetahat_star <- lapply(
  X = x_star,
  FUN = mean
)
thetahat_star <- as.vector(
  do.call(
    what = "rbind",
    args = thetahat_star
  )
)

\(\mathrm{se} \left( \hat{\theta}^{*} \right)\)

The standard error of the bootstrap sampling distribution \(\hat{\theta}^{*}\) is given by

\[\begin{equation} \mathrm{se} \left( \hat{\theta}^{*} \right) = \sqrt{ \frac{1}{B - 1} \sum_{b = 1}^{B} \left[ \hat{\theta}^{*} \left( b \right) - \hat{\theta}^{*} \left( \cdot \right) \right]^2 } = 0.1172739 \end{equation}\]

where the mean of the bootstrap sampling distribution \(\hat{\theta}^{*}\) is given by

\[\begin{equation} \hat{\theta}^{*} \left( \cdot \right) = \frac{1}{B} \sum_{b = 1}^{B} \hat{\theta}^{*} \left( b \right) = 0.0224161 . \end{equation}\]

The standard error can be used to make inferences about \(\hat{\theta}\).

Confidence Intervals

Confidence intervals can be constructed to make inferences about the stability of \(\hat{\theta}\).

wald <- wald(
  thetahat = muhat,
  se_thetahat = se_muhat,
  theta_null = 0,
  alpha = c(
    0.001,
    0.01,
    0.05
  ),
  distribution = "z"
)
pc <- pc(
  thetahat_star = thetahat_star,
  alpha = c(
    0.001,
    0.01,
    0.05
  ),
  wald = TRUE,
  thetahat = muhat,
  theta_null = 0,
  distribution = "z"
)
bc <- bc(
  thetahat_star = thetahat_star,
  thetahat = muhat,
  alpha = c(
    0.001,
    0.01,
    0.05
  ),
  wald = TRUE,
  theta_null = 0,
  distribution = "z"
)
bca <- bca(
  thetahat_star = thetahat_star,
  thetahat = muhat,
  data = x,
  fitFUN = mean,
  alpha = c(
    0.001,
    0.01,
    0.05
  ),
  wald = TRUE,
  theta_null = 0,
  distribution = "z"
)
